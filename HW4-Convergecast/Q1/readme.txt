The program finds the max value of a given graph using converge cast. It aslo appends all the values of child nodes and creats a single valued string at the root node.

There are two options to run the program:
First -
1. Extract the zip file
2. Import the extracted project as an existing java project in eclipse
3. Run

Second - 
1. Keep both the Processor.java and Main.java in the same directory
2. Open terminal in that directory
3. Run command - javac *.java
4. Run command - java Main

The output will be printed on the console.
